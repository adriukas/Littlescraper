# ProjectLib — Discord Vinted Scraper

A Laravel web application that monitors Discord channels for Vinted item purchase activity. Users can manage multiple Discord bots, trigger scrapes, and browse sales/message history.

---

## Features

- Add, edit, and delete Discord bots (token + channel ID)
- Scrape Discord channel messages via a Python script
- View and manage sales history and message history
- Simple login system with credentials defined in `.env`

---

## Tech Stack

- **Backend:** Laravel 11 (PHP 8.4)
- **Frontend:** Blade templates
- **Database:** MySQL 8.0
- **Scraper:** Python 3 (`script/littlescraper.py`)
- **Server:** Apache (inside Docker)

---

## Requirements

- Docker and Docker Compose (or Podman and podman-compose)
- Git

---

## Local Setup

```bash
git clone <repo-url>
cd ProjectLib

cp .env.example .env
# Edit .env — set DB_PASSWORD, DISCORD_TOKEN, ADMIN_EMAIL, ADMIN_PASSWORD

docker compose up -d --build

docker compose exec app php artisan key:generate
docker compose exec app php artisan migrate --force
docker compose exec app php artisan db:seed
```

App will be available at **http://localhost:8000**

## MIF Cloud Deployment

### 1. Create a VM
Log in at [mifcloud.mif.vu.lt](https://mifcloud.mif.vu.lt), create an Ubuntu instance, and add your SSH key.

### 2. Connect and install Docker
```bash
ssh ubuntu@<vm-ip>
sudo apt update && sudo apt install -y docker.io docker-compose
sudo usermod -aG docker $USER && newgrp docker
```

### 3. Clone and configure
```bash
git clone <repo-url>
cd ProjectLib
cp .env.example .env
nano .env   # fill in all required values
```

### 4. Build and run
```bash
docker compose up -d --build
docker compose exec app php artisan key:generate
docker compose exec app php artisan migrate --force
docker compose exec app php artisan db:seed
```

### 5. Open port in MIF Cloud
In the MIF Cloud dashboard, add a security group rule to allow **TCP port 8000** inbound.

App will be available at **http://\<vm-ip\>:8000**

## Environment Variables

Copy `.env.example` to `.env` and fill in:

| Variable | Description |
|---|---|
| `APP_KEY` | Generated by `php artisan key:generate` |
| `DB_PASSWORD` | MySQL root password (must match `docker-compose.yml`) |
| `DISCORD_TOKEN` | Bot token from Discord Developer Portal |
| `ADMIN_EMAIL` | Login email for the admin account |
| `ADMIN_PASSWORD` | Login password for the admin account |

> **Never commit your `.env` file.** It contains secrets.
